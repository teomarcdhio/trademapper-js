TradeMapper is an interactive browser-based tool developed by TRAFFIC and WWF to visualise wildlife trade data quickly and easily.

Trade data

To get started, upload your data in CSV format.

Map layers

Add up to three layers to your map in TopoJSON format. Convert map data at mapshaper.

You can export the current view for use in other applications. Choose your preferred format below.

SVG

SVG is a scalable image which can be imported by most common applications.

Animated GIF

Use this format if you have data spanning multiple years. Your map will be animated and can be shared across social media applications.
